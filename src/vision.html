The main goal of Gatherspiel is to promote human connection through board games and promote public third spaces. To promote this goal, these guidelines wil be followed for any changes.

Support for in person interaction

All content added to Gatherspiel will support the goal of encouraging people to interact in person. For example, a feature that makes it easier for people to find in person events would be a good fit for the website. Changes to improve usability or readability of the website also support this goal. Forums, comments, or any other form of online messaging will not be supported by Gatherspiel. While these are valuable for developing and growing a board game community, there are also many other platforms that prioritize these types of communication.

Content is human generated

As a website that promotes human connection, Gatherspiel will be generated by humans without the use of generative AI. This includes not using generative AI for art, text, or code.

All users see the same content

All users will see the same content, and Gatherspiel will not provide personalized recommendations. Gatherspiel encourages getting recommendations through connecting with others in person at events. When users see events or other content on Gatherspiel, it should be because they made a decision to view that content.

Non Commercial

Gatherspiel will not be a profit making entity. Monetization will be minimized and only be used for the goal of supporting expenses associated with the platform.

Decentralization

While hosts can share information about their events using Gatherspiel's platform, Gatherspiel will also support hosting events using other platforms. Data stored on Gatherspiel will be able to be accessed and exported ActivityPub powered API. All code used for the Gatherspiel platform will also be open source and will be able to be used by people looking to self host.
